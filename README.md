Hoodworks App is a community-driven marketplace that connects nearby residents to buy, sell, and exchange goods and services without leaving their homes. It combines local trading, task outsourcing, and real-time communication into one simple platform.

Users can list items for sale, request services, or post tasks they need help with—everything from selling a phone to finding someone nearby for quick repairs. A built-in map allows users to pin their exact location, making it easy for others to find them quickly and coordinate efficiently within the same neighborhood.

The app includes a chat system where users can negotiate prices, ask questions, and finalize agreements instantly. By focusing on proximity, Hoodworks builds trust, reduces delivery time, and strengthens local connections, turning neighborhoods into self-sustaining, digitally connected communities.
